【给 Code 的使用说明 · 出厂前删除本块】
1. 这是可运行的三层骨架空壳。你只需把 {{槽位}} 换成正式数据源里的真值，不要重新设计结构或配色。
2. 所有 {{}} 都是占位符、故意不写成像真数的数字——防止像上次那样被误当成正式输入。凡 {{...}} 未替换即视为未完成，出厂 lint 应拦。
3. 每只股票在第二层、第三层各循环一张卡；三层的核心字段必须全部读自同一个 final_decision 对象，本模板不做任何计算。
4. 图表用 <div class="chart" data-chart="N"> 占位，已注明每张图的数据来源与必需元素；画法自选（CSS条/SVG均可），但必须有一句大白话结论 + 数据日期 + 来源。
5. 手机端已内置响应式（表格自动转卡片）：给 <td> 加 data-l="列名" 即可在手机上显示列名。
6. 颜色变量在 :root，不要改；动作色必须配文字+图标，不得只靠颜色。
