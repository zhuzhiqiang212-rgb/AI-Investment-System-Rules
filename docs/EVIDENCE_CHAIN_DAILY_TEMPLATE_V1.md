## 二、力度打分口径
强=有硬证据（政策文件/官方数据/明确盘面）且方向一致；中=有信号未确认或证据混合；弱=只有猜测传闻或证据矛盾；证伪=出现反向硬证据（如美联储明确恢复全球普惠放水、或明确否认围绕战略滴灌）。

## 三、自动推导规则（机器核心）
总闸力度决定机会池口径：
- 总闸=强 → 机会池全口径（美国AI战略供应链全节点可进池），仓位正常。
- 总闸=中 → 收口径（只留最硬核心节点：算力/设备/代工龙头），偏谨慎。
- 总闸=弱 → 大幅收口（只守现有核心、不新增机会），降风险。
- 总闸=证伪 → 停用战略供应链口径，切普通AI牛市备用逻辑，显著降风险。
方向规则：总闸成立且战略指向AI → 今日方向=做多美国AI战略承接节点，机会池按上表口径；总闸成立但战略某环弱 → 方向保留、机会池收一档；总闸证伪 → 不走战略链、转备用逻辑。
