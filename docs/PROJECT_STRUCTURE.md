# PROJECT_STRUCTURE

## 顶层目录

- `docs/`：系统说明、交接文档、GitHub/Google Drive 分工、角色分离规则。
- `skills/`：AI 岗位说明书草案。
- `scripts/`：未来可迁移到 GitHub 的通用脚本目录。
- `templates/`：日报、验收包、HTML 卡片等模板目录。
- `rules/`：系统规则、质量目标、数据缺口规则。
- `system/`：任务状态、仪表盘、系统脚本、状态 JSON。
- `reports/`：Google Drive 运行产物，包括日报、验收包、历史归档。
- `data/`：行情、市场、数据源说明。

## 用户入口

用户每天从 `START_HERE.html` 进入，不直接从 GitHub 阅读日报。
