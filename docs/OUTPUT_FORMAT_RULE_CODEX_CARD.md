# 输出格式规则 — Codex执行卡
# 生效时间：2026-06-17 JST
# 状态：永久有效

## 规则

所有给Codex的指令，必须做成一键复制卡片。

### 格式要求

1. 做成交互式HTML卡片，带「全部复制」按钮
2. 点击按钮，自动复制全部内容到剪贴板
3. 用户直接粘贴给Codex，无需额外操作
4. 卡片内包含：任务说明 + 完整脚本 + 执行命令 + 验收确认项
5. 不需要下载文件，不需要分辨哪段是说明哪段是代码

### 触发条件

只要需要发给Codex执行的内容，无论长短，都用此格式。
包括：脚本、配置修改、文件创建、路径迁移、任何系统操作。

### 禁止格式

禁止：只输出markdown代码块让用户自己复制
禁止：输出多个文件让用户分辨哪个发给Codex
禁止：让用户手动拼接多段内容
