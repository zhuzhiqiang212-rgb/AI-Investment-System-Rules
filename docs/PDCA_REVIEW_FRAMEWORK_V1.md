# PDCA_REVIEW_FRAMEWORK_V1：复盘位框架

> 状态：v1 骨架。只建字段位，不自动复盘，不改日报，不下单，不发布。

## 1. 次日复盘
- 复盘对象：昨日判断是否正确，昨日禁止动作是否守住，昨日风险是否兑现。
- 字段位：昨日判断 / 今日实际 / 对错 / 错在体系哪一步 / 明日修正。
- 接入能力档案：把错因写入岗位能力档案的教训清单。

## 2. 周报复盘
- 复盘对象：一周内判断连续性、风险识别、动作纪律。
- 字段位：本周正确判断 / 本周错误判断 / 最大漏项 / 下周改进。
- 接入 PDCA：把重复错因升级为流程或闸门修正。

## 3. 月报复盘
- 复盘对象：月度收益、风险暴露、研究理解是否有效改变动作。
- 字段位：收益归因 / 风险归因 / 研究理解命中 / 需要升级的岗位能力。
- 接入能力档案：沉淀为经验库与能力指标。

## 4. 季报复盘
- 复盘对象：赛道配置、估值纪律、组合结构是否符合大方向。
- 字段位：赛道胜负 / 配置偏差 / 估值偏差 / 下一季制度修正。
- 接入 PDCA：必要时调整 SOP、验收标准或质量闸门。

## 5. 年报复盘
- 复盘对象：公司能力是否成长，是否持续生产高质量投资决策。
- 字段位：年度能力进步 / 年度最大系统错误 / 已淘汰资产 / 新增岗位能力。
- 接入能力档案：形成下一年组织升级清单。
