# 非战略持仓简易估值说明 V1

【总则对照】对照六条：第一条→本说明按商业模式归类，不从持仓反推；第二至五条→不改证据链骨架、求证表、驱动、复盘；第六条防静态化→自评：这里只定义非战略持仓的简易估值处理规则，分派器按商业模式锚动态识别，可迭代升级，不锁死永久答案。

## 一、定位
这批标的不在当前 AI 战略主线内，估值处理目标不是建立深度模型，而是给受检持仓一个诚实的简易估值框架。精力优先给战略标的与已激活机会池。

## 二、六只标的归类

| 标的 | 商业模式类型 | 分派器分类 | 为何不做深度估值 | 升级深做条件 |
|---|---|---|---|---|
| 任天堂 | 消费娱乐 IP | 非战略·简易PE | 非 AI 战略主线，先用内容周期与简易 PE 观察即可 | 新硬件周期或 IP 资产与战略链发生明确连接 |
| 索尼 | 综合电子与娱乐分部 | 非战略·分部 | 业务多元，当前不是战略主线承接节点，先做分部简化观察 | 半导体、AI 影像或内容平台成为战略相关核心变量 |
| 丰田 | 汽车周期 | 非战略·周期 | 汽车周期与汇率敏感，不属于当前 AI 战略主线 | 自动驾驶、能源或产业政策使其进入激活节点 |
| 万代南梦宫 | 消费娱乐 IP | 非战略·简易PE | 内容周期股，当前只需简易 PE 与内容 pipeline 观察 | 核心 IP 或平台能力进入战略机会池 |
| 伊藤忠 | 综合商社 | 非战略·PB股息 | 商社更适合 PB、股息、资源周期简评，不需占用深度模型资源 | 资源、能源、安全链条被证据链激活 |
| IBKR | 券商 | 非战略·PE | 极小仓且非战略主线，简易 PE 与交易活跃度观察即可 | 若金融基础设施成为证据链激活节点再升级 |

## 三、分派规则
valuation_dispatcher.py 在深度模型识别之后、待人工归类兜底之前，按汽车、消费娱乐 IP、综合电子、综合商社、券商等商业模式锚归入“非战略简易估值”。

## 四、边界
非战略简易估值不是永久忽略。若证据链或公司业务变化使某只标的转为战略相关，允许升级为正式模型实例或新增模型类型。
