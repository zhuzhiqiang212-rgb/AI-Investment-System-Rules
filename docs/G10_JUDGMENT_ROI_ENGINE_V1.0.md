# 日期：2026-06-17 JST

# 状态：待V6验收

---

## 核心定义

判断ROI ≠ 个股ROI

个股ROI：NVDA +76%
判断ROI：「识别AI算力为核心主线并持有」这个判断的ROI是多少？

区别：
同一只股票，不同的判断可能产生完全不同的ROI。
例如NVDA：
  - 判断A：「AI算力主线，长持」→ +76%，ROI高
  - 判断B：「FOMC前减仓规避风险」→ 踏空，ROI低
  - 判断C：「FOMC偏鸽后加仓」→ 待验证

---

## 判断ROI计算公式

判断ROI = (判断对应的持仓市值变化) / (判断触发时的持仓成本)
权重调整 = × (该持仓占总组合的比例)
加权判断ROI = 判断ROI × 仓位权重
