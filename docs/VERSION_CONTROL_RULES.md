# VERSION_CONTROL_RULES

## 本地优先

当前阶段只建立本地 GitHub 项目结构，不推送远程，不要求用户注册 GitHub。

## 可进入 GitHub 的内容

- README.md
- docs/
- skills/
- rules/
- scripts/
- templates/
- system/*.md
- system/*.json

## 不进入 GitHub 的内容

- 账户截图
- PDF / 视频 / 音频
- reports/daily/
- reports/history/
- reports/validation/
- inbox/
- inputs/
- 隐私资料

## 版本提交前检查

提交前必须确认没有账户截图、隐私资料、历史日报、验收包或大型文件。
