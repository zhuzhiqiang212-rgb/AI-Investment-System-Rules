## 一、主输出：《AI投研日报》PDF

### 定位
用户每天查看的唯一文件。
可直接微信发送，可手机阅读。

### 文件规范
文件名: AI投研日报_YYYY-MM-DD.pdf
输出路径: C:\Users\zhu20\OneDrive\桌面\股票分析与研究\
覆盖规则: 同日期版本自动覆盖，不产生重复文件
历史保留: 不同日期版本保留，供历史查阅
排序: 按日期命名，新文件自动排在最前面

### PDF内容结构（单页，手机可读）

• [第2层第1条]
• [第2层第2条]

━━ 持仓状态 ━━
[四账户真实持仓简表，含浮盈%]

[当周关键事件提示]

━━ 下一步 ━━
[用户今日需要关注的唯一一件事]

⚠️ 仅供参考，不构成投资建议

## 二、内部文件（系统使用，不向用户主动展示）

以下文件允许存在于系统，但不作为用户主要查看对象：
  方向卡（direction_card）
  执行卡（execution_card）
  风险卡（risk_card）
  latest_* 系列文件
  中间计算文件

用户如需查看内部文件，可按需提供，但默认不展示。

---

## 三、知识交付标准

禁止仅输出: Drive ID / 文件编号 / 内部路径
必须满足以下至少一种:
  1. GitHub链接
  2. GitHub Raw链接
  3. Markdown正文（在对话中直接输出）
  4. 可直接阅读的PDF文件

否则视为未完成交付。
