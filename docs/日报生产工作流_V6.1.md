## 完整工作流

  检查 Drive「投研资料库」最新更新
  检查 Drive「股票分析与研究」前日日报
确认: 今日有无重大市场事件需纳入日报

### 步骤2：GitHub同步检查（恢复项）

执行人: Claude
内容:
  确认 knowledge/ 目录最新版本与Drive一致
  确认 auto_briefing.py 版本正确
  若有不一致，发起同步任务后再生成日报

### 步骤3：知识库一致性检查（恢复项）

执行人: Claude
内容:
  确认 01_INVESTMENT_RULES.md 无未生效变更
  确认 02_REPORT_STANDARD.md 为最新版
  确认当日工作流无未完成的治理任务

### 步骤4：数据获取

执行人: Codex（auto_briefing.py）
数据源:
  VIX → Yahoo Finance（^VIX）
  10Y美债 → Yahoo Finance（^TNX）
  SPX日涨跌 → Yahoo Finance（^GSPC）
  BTC日涨跌 → Binance API（BTCUSDT）
  富途持仓 → 富途OpenD（127.0.0.1:11111，只读）
  加密仓位/月收益 → user_config.json
输出: 结构化数据字典

### 步骤5：周期判断与结论生成

执行人: Codex（auto_briefing.py）
内容:
  4维周期判断（VIX/美债/SPX/BTC）
  置信度评级（A/B/C）
  三层结论生成（第0/1/2层）
  昨日验证（对比auto_briefing_log.json）
规则:
  TRANSITION/UNKNOWN/BEAR → 强制"观察"
  BULL_MID A级 → 允许"分批建仓"
  第2层至少两条，第二条含当日具体数据

### 步骤6：用户确认

执行人: 用户
方式: 输入 Y 确认，输入 N 取消
禁止: 跳过用户确认直接写入

### 步骤7：PDF日报生成（新增项）

执行人: Codex（generate_pdf_report()）
文件名: AI投研日报_YYYY-MM-DD.pdf
内容:
  市场数据栏（VIX/10Y/SPX/BTC）
  今日结论（周期+置信度+三层结论）
  禁止事项
  富途实时持仓
  SBI日股持仓
  本周节奏
  今日必做提醒
页面规格: A4单页，手机可读，中文字体

### 步骤8：写入桌面（新增项）

执行人: Codex
写入路径: C:\Users\zhu20\OneDrive\桌面\股票分析与研究\
文件名格式: AI投研日报_YYYY-MM-DD.pdf
覆盖规则: 同日期自动覆盖，不产生重复文件
排序: 按日期命名，新文件自动排在最前面
同时写入:
  Drive reports/daily/（存档）
  Drive 股票分析与研究文件夹（云端备份）

### 步骤9：微信发送格式（新增项）

执行人: Claude（按需生成）
触发: 用户要求发微信时
格式: 单个代码框，一次复制，包含完整日报内容
禁止: 多个复制按钮 / 要求用户分段复制

### 步骤10：日报入口层设计（新增项）

用户查看入口（唯一）:
  桌面 → 股票分析与研究 → AI投研日报_YYYY-MM-DD.pdf
  打开方式: WPS PDF（默认）
  特点: 新文件排最前，无需搜索
