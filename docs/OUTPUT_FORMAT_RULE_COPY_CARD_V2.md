# 输出格式规则 — 一键复制卡片
# 生效时间：2026-06-17 JST
# 状态：永久有效

## 规则

所有发给第三方执行的内容，必须做成一键复制卡片。

### 适用对象

- Codex（执行脚本/系统操作）
- ChatGPT V6（审批/方向指引）
- 任何需要复制粘贴给其他工具的内容

### 格式要求

1. 做成交互式HTML卡片，带「全部复制」按钮
2. 点击按钮，自动复制全部内容到剪贴板
3. 用户直接粘贴，无需额外操作
4. 卡片顶部标注：发给谁（Codex / GPT / 其他）
5. 不需要下载文件，不需要分辨哪段是说明哪段是内容

### 卡片顶部标注示例

发给 Codex → 标注「Codex 执行卡」
发给 GPT   → 标注「ChatGPT V6 审批卡」
发给用户自己参考 → 标注「参考卡（无需发送）」

### 禁止格式

禁止：只输出markdown代码块让用户自己复制
禁止：输出多个文件让用户分辨发哪个
禁止：让用户手动拼接多段内容
禁止：不标注发给谁
