# GOOGLE_DRIVE_VS_GITHUB

## Google Drive

Google Drive 是资料和报告仓库，保存每日运行文件：日报、截图、输入资料、历史报告、验收包、账户快照。

## GitHub

GitHub 是系统代码保险柜，保存脚本、规则、Skill、模板、说明文档、目录结构和质量标准。

## 分工原则

- Google Drive 存用户资料和每天产物。
- GitHub 存可迁移、可版本管理的系统结构。
- 验收包和日报不作为 GitHub 主体内容。
- 任何工具接手前先读 README、CURRENT_SYSTEM_STATUS、ROLE_SEPARATION_RULES 和 Skill 文件。
