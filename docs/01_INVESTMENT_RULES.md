# AI投研明鉴规则库 V6

版本: V6.0
生效时间: 2026-06-14 JST
最终目标: 四账户年化收益率 40%+
账户原则: 只有用户本人可以下单和出入金

## 一、仓位管理规则

### 硬上限（任何周期下不可突破）
| 资产类别 | 硬上限 |
|---------|-------|
| 美股（IBKR） | 总资产 50% |
| 日股（SBI） | 总资产 30% |
| A股（富途） | 总资产 30% |
| 加密（BF） | 总资产 10% |
| 现金/避险 | 最低 10% |

### 仓位规则
- 任何单一标的不超过账户净值 20%
- 同一方向信号不超过账户净值 50%
- 禁止满仓（任何账户仓位上限 80%）
- 禁止使用杠杆超过账户净值
- 回撤>10% 时仓位强制压缩至 50%

### 信号等级与仓位对应
| 信号等级 | 建议仓位 |
|---------|---------|
| A级 | 标准仓位（按引擎计算） |
| B级 | 标准仓位 × 0.7 |
| C级 | 禁止新建仓位 |

## 二、风险管理规则

### 止损规则
- 单笔交易最大亏损：账户净值 3%
- 触发止损后当日不再新建同方向仓位
- 止损价位在进场时确定，不事后移动

### EMERGENCY MODE 触发条件（任意一条）
- 全球主要股指单日跌幅 > 5%
- VIX 单日跳升 > 30%
- 用户手动宣布 EMERGENCY

### EMERGENCY MODE 规则
- 周期标签自动降级至 BEAR
- 所有买入触发器暂停
- 现金比例强制提升至 30%
- 解除条件：连续5个交易日无新增触发且用户手动确认

### 年化目标边界
- 允许落后目标时提高风险敞口，但不超过基准配置+10%
- 绝对禁止：杠杆超净值 / 突破资产硬上限
- 绝对禁止：TRANSITION/UNKNOWN/EMERGENCY期间提高敞口

## 三、买卖纪律

### 买入条件（全部满足才能买）
1. 周期标签为 BULL_MID 或 BULL_EARLY
2. 置信度 A 级（B级可小仓位试探）
3. 信号等级 A 或 B
4. 账户仓位未达上限
5. 用户人工确认

### 不买入条件（任意一条禁止买入）
- 置信度 C 级
- 周期为 TRANSITION / UNKNOWN / BEAR
- 标的5日涨幅 > 12%
- 账户回撤 > 10%
- EMERGENCY MODE 触发中
- 数据获取失败≥3项

### 止盈规则
- 第一目标：进场价 +15%，止盈 50% 仓位
- 第二目标：进场价 +30%，止盈剩余 50%
- 周期失效时：不论盈亏强制止盈
- 永不止盈资产：无（所有资产均有止盈）

### 不止盈条件
- 仓位<3%（太小，摩擦成本高于收益）
- 距第一目标<5%（等待达到再止）

## 四、周期分析规则

### 四维判断（VIX / 10Y美债 / SPX / BTC）
| 维度 | BULL信号 | TRANSITION | BEAR信号 |
|------|---------|-----------|---------|
| VIX | <18 | 18-23 | >23 |
| 10Y美债 | <4.0% | 4.0-4.7% | >4.7% |
| SPX日涨跌 | >+0.5% | -1%~+0.5% | <-1% |
| BTC日涨跌 | >+1% | -2%~+1% | <-2% |

### 置信度规则
| 一致性 | 置信度 |
|-------|-------|
| 4/4维一致 | A |
| 3/4维一致 | A |
| 2/4维一致 | B |
| 1/4维一致 | C |
| 数据不足<3维 | 降一级 |

### 周期失效条件
| 周期 | 失效条件 |
|------|---------|
| BULL_MID | VIX突破25 / SPX单日跌>3% |
| TRANSITION | VIX突破30 / 关键支撑跌破 |
| BEAR | VIX回落<20且SPX连续3日上涨 |

## 五、归因原则

### 模块责任分类（六类，禁止笼统归因）
1. 周期定位错误
2. 资产配置错误
3. 买入触发错误
4. 仓位计算错误
5. 止盈规则错误
6. 用户执行偏离

### 最低样本规则
| 归因类型 | 最低样本 |
|---------|---------|
| 触发器各等级胜率 | 10次 |
| 周期标签有效性 | 8次 |
| 单账户月度归因 | 5笔 |
| 模块责任归因 | 3次同类错误 |

样本不足时禁止输出任何确定性结论，只输出：
"样本不足（N次/最低M次），继续观察"

### 未交易归因（必须记录）
- 类型A：触发信号但未买入
- 类型B：应减仓但未减仓
- 类型C：不应买入但买入了
- 类型D：应止盈但未止盈
