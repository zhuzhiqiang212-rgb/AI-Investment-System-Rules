## 0. 这份标准的魂（最高原则）

**日报不按"几页"定合格，按"能不能说服董事长"定合格。**

- 短了说不服 = 不合格；长了废话 = 不合格。
- 一切以"看完点头：嗯，今天该这么干"为准。
- 全程大白话、图文并茂、对比着讲。

系统整体分两大件：
- **第一件：每日决策简报** —— 天天出、天天更新，负责"说服你今天怎么做"。
- **第二件：深度研究底座** —— 不天天出、按需更新，负责"扛证据"。
- 关系：简报里每个"为什么"，都能往下钻到底座看到完整推导。**简报负责说服，底座负责扛证据。**

# 第一件 · 每日决策简报（天天出）

> 不限页数。原则：大白话 + 图文并茂 + 对比着讲 + 每个动作都能说服人。
> 数据不实时必须明说"仅参考、不能下单"；缺数据老实标"缺"，不装懂；不自动下单。

## 简报结构

### 板块一：今天大局（第一屏）
一眼看完今天整体。用大白话回答：
- 今天动不动？（不主动交易 / 有动作）
- 今天最大风险要不要今天处理？
- 今天要盯哪几个？
- 今天有没有出现更好的公司？有没有该退的？
- 一句话结论：今天到底怎么干。

### 板块二：每只持仓今天怎么办（逐仓）
每只持仓给一组"能说服你"的说明，不是干巴巴结论：

1. **今天这么做**（持有/等待/复核/不追…）
2. **为什么**（大白话 + 对比：为什么是这个动作，不是另一个）
3. **🔴 但反过来想**（反方：如果我错了会错在哪 + **翻盘信号**：什么信号一出现，今天这个判断就要重新评估）
4. **什么时候改变**（触发线：到什么价、什么条件动作会变）
5. **最容易犯的错**（提醒你别踩的坑）

> **硬规矩：每只重要持仓必须带「🔴但反过来想 + 翻盘信号」。缺这一栏 = 不合格，打回。**

### 板块三：今天最大的风险
- 用对比和图示讲清：最大风险是什么、影响哪些资产、今天要不要动手。
- 集中度、加密波动、估值链路等，哪个是今天真问题。

### 板块四：今天最大的机会（观察池 Top 3）
- 不是让你买，是"值得盯的好资产 + 到什么价/什么条件才动手"。
- 每个候选：为什么进前三、为什么现在还不买、到什么线才复核。

### 板块五：研究理解（补简版最大的缺口）
今天的资料（老雷/湖水/Drive/会议纪要等），大白话回答：
- 哪几条**真改变了今天的判断**？（改了什么、为什么）
- 哪几条只是**背景**？（知道就行，不改动作）
- 哪几条是**噪音**？（不用管）
- **硬规矩**：缺的研究老实标"缺/待补"，不拿旧的、空的冒充。

### 板块六：证据链 + 明日 PDCA
- 每个重要结论背后：支撑证据、反向证据、失效条件。
- 明天要复核什么（今天判断对不对，明天怎么验证）。

## 每日简报质量红线（不达标打回）
1. 每个重要动作必须能"说服"——有为什么、有对比。
2. 每只重要持仓必须带"但反过来想 + 翻盘信号"。
3. 行情非实时必须标注，不得用于下单结论。
4. 缺数据必须标"缺"，禁止用旧快照/空值冒充。
5. 不自动下单，买卖动作留给董事长确认。
6. 全程大白话，禁止术语堆砌、后台字段（路径/哈希/日志）直出正文。

# 第二件 · 深度研究底座（不天天出，按需更新）

> 可以很厚（100-200页不限）。但必须：大白话 + 分部分 + 部分之间逻辑清楚、证据硬。
> 它是每日简报背后的"为什么"仓库，简报的结论都能钻到这里看到完整推导。

## 底座分三部分（部分之间逻辑递进）

### 底座① 世界怎么了（宏观底座）
- 利率、流动性、美元、政策、地缘——决定大方向。
- 大白话讲清：现在世界处于什么阶段，对我的资产是顺风还是逆风。
- 输出：当前宏观大方向判断 + 失效条件。

### 底座② 每个赛道和公司值多少（价值底座）
- 各赛道（AI/加密/日股/防御…）+ 每只持仓公司的估值、护城河、财报三指标。
- 大白话讲清：谁贵谁便宜、谁的生意好、谁值得长期拿。
- 输出：估值分级 + 每家公司"值不值"的判断。

### 底座③ 机会与风险全景（机会底座）
- 全市场扫描出的好机会 + 你组合的整体风险全貌。
- 大白话讲清：该盯谁、该防谁、组合哪里过重。
- 输出：机会清单 + 风险全景 + 该退/该盯清单。

### 逻辑链（三底座如何递进）
① 世界大方向 → 决定 ② 哪些赛道公司此刻值得 → 汇成 ③ 具体盯哪些机会、防哪些风险。
每日简报的每个结论，从这三层各取一块证据，拼成"为什么"。

## 重大决策的"反方挑战单"（留档在底座）
遇到重大动作（真买入/卖出/大幅调仓），除简报里的"但反过来想"外，另出一份系统性反方单：
- 这个决定最可能死在哪？
- 有没有我们集体忽略的风险？
- 什么证据出现，就该推翻这个决定？
- 反方单随重大决策留档，供 GPT V6 复审。

## 深度底座质量红线
1. 必须大白话，任何专业概念都要翻译成人话。
2. 分部分，部分与部分之间逻辑必须递进、不跳步。
3. 每个结论必须有证据，证据力度要够（不能"我觉得"）。
4. 缺数据标"缺"，不伪装完整。
