## 二、每日决策简报必须包含的板块（照 V7.2）

1. 今天大局（第一屏）：动不动、最大风险要不要今天处理、盯谁、有无更好/该退的、一句话结论。
2. 今天发生了哪些事：Claude 过滤后的今日大事；宏观归宏观；老雷/湖水只当辅助佐证，不单独作结论依据。
3. 每只股票今天怎么办（5件套）：今天这么做 / 为什么(对比) / 🔴但反过来想(错在哪+翻盘信号) / 什么时候改变 / 最易犯错。
4. 日股单独分析：日股(软银/任天堂等)单独一块，每天出，不与美股混；含日元/BOJ因素、日股专属风险机会。
5. 今天最大的风险。
6. 机会与板块轮动（分短/中/长期）：见第三节硬要求。
7. 研究理解：哪条真改判断 / 哪条背景 / 哪条噪音；缺的标"缺"。
8. 证据链 + 明日复核。

## 三、本次重点硬要求（缺任一即不合格）

【硬要求1｜反调】每只重要股票必须带"🔴但反过来想 + 翻盘信号"。

【硬要求2｜机会池买卖计划表】每个进正文的机会，必须给全：
  - 信号等级 A/B/C（沿用 docs/buy_trigger_engine_v1.md 的三级 + 四关过滤：方向/估值/风险/账户确认）
  - 深度研报理由（为什么是它，有研究支撑）
  - 💰 具体买入价（数字）
  - 📈 具体止盈价（数字）
  - 🛑 具体止损价（数字）
  给不出三个价 = 不合格。机会分短/中/长期三档呈现。

【硬要求3｜账户与现金】
  - 富途：OpenD 读取，标"较新"；**基金栏金额计入可用现金**。
  - IBKR/SBI/bitFlyer：手动截图来源，必须标"截图X月X日，有新交易以实际为准"，不拿旧截图当今天真实持仓。

【硬要求4｜诚实与大白话】
  - 行情非实时必须标注，不得用于下单结论。
  - 缺数据必须标"缺"，禁止用旧快照/空值/固定折扣占位冒充。
  - 全程大白话，禁止术语堆砌、禁止后台字段(路径/哈希/日志)直出正文。
  - 绝不自动下单。
