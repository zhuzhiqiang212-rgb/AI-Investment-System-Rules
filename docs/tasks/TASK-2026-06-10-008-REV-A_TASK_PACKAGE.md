## 审批变更项（已合并，共4项）

1. 新增"不确定周期"状态
   在原有四阶段基础上新增：
   TRANSITION（周期过渡期，数据冲突/周期切换中）
   UNKNOWN（证据不足，禁止强行归类）
   共六个周期状态。

2. 新增周期置信度
   每次周期判断必须输出置信度等级：
   A级（高置信）/ B级（中置信）/ C级（低置信）
   防止弱证据包装成强结论。

3. 新增资产类别差异
   禁止一个周期标签覆盖全部资产。
   必须定义总周期（宏观流动性周期）
   与资产子周期（AI股/加密/日股/A股各自周期）的关系。

4. 新增周期失效条件
   每个周期阶段标签必须附带失效条件，
   否则周期判断无法复盘。

2. 六个周期状态定义与判断矩阵（变更项1）
   须定义：
   - BULL_EARLY（牛市初期）
   - BULL_MID（牛市中期）
   - BULL_LATE（牛市末期）
   - BEAR（熊市/调整期）
   - TRANSITION（过渡期）
   - UNKNOWN（证据不足）
   每个状态须包含：
   - 判断指标与阈值
   - 置信度判断规则（变更项2）
   - 周期失效条件（变更项4）
   - 对应仓位策略基准
   - 允许/禁止执行的操作
   - 典型持续时间参考

3. 周期置信度输出格式（变更项2）
   A级：多数指标一致，无重大矛盾信号
   B级：指标分歧，但主要方向明确
   C级：指标严重分歧，或关键数据缺失
   置信度 C级时：仓位自动降至 B级仓位上限

4. 总周期与资产子周期框架（变更项3）
   总周期：宏观流动性周期（美联储/全球M2/美债）
   资产子周期（独立定义）：
   - AI股子周期（AI资本开支周期/叙事强度）
   - 加密子周期（BTC半减期/链上周期/MVRV）
   - 日股子周期（日银政策周期/汇率周期）
   - A股子周期（政策催化周期/北向资金周期）
   每个资产子周期须定义：
   - 与总周期的关系（同步/领先/滞后/独立）
   - 子周期独立判断指标
   - 子周期与总周期冲突时的处理规则

5. 周期失效条件清单（变更项4）
   每个周期状态标签须附带：
   - 数据失效触发条件（指标越过阈值）
   - 事件失效触发条件（黑天鹅/政策突变）
   - 时间失效触发条件（超过典型持续时间上限）
   - 失效后的默认降级状态

6. 周期判断指标清单（六维）

   - AI叙事（资本开支/研发投入/叙事强度）
   - 加密周期（BTC半减期/链上数据/MVRV）

7. 四账户周期联动规则
   每个账户在六个周期状态下的：
   - 仓位基准调整系数
   - 触发器参数调整

   - TRANSITION/UNKNOWN状态下的保守策略

8. 日报 V2 周期状态字段格式
