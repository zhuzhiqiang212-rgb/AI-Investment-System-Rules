# Buy Trigger Engine V1

来源任务：TASK-2026-06-10-005 REV-A
部署任务：TASK-2026-06-10-017
用途：Daily Briefing V2 查表区步骤3，判断是否出现买入触发信号。

## 1. 触发器设计原则

买入触发器只输出信号等级和人工确认事项，不生成交易指令，不自动下单。
任何触发信号都必须同时通过：
- 市场方向。
- 估值条件。
- 风险卡。
- 账户人工确认。

## 2. A/B/C 信号等级定义

| 等级 | 定义 | 允许动作 | 禁止动作 |
|---|---|---|---|
| A | 多项证据一致，方向/估值/风险均支持 | 可进入仓位引擎计算建议比例 | 禁止跳过账户确认直接下单 |
| B | 方向基本支持，但估值或风险仍有分歧 | 仅允许小仓观察或等待确认 | 禁止重仓、禁止追高 |
| C | 证据不足或信号冲突 | 只观察 | 禁止买入 |

## 3. 四账户各自触发条件模板

### 富途账户模板

- 买入触发条件：核心AI或A股方向进入可观察区间，风险卡未触发高拥挤，账户现金/挂单已人工确认。
- 不买入条件：高beta过热、估值偏贵且趋势脉冲、账户字段待确认、已有重复风险暴露。
- 仓位上限：由position_sizing_engine_v1.md计算，不得突破账户硬上限。
- 失效条件：方向卡转弱、风险卡升级、估值脱离观察区间、账户现金不足。

### IBKR账户模板

- 买入触发条件：美股核心资产回调但长期主线未破，ETF Flow/AI Narrative未恶化。
- 不买入条件：NVDA/COIN/MSTR等高beta冲高、Gamma/CTA过热、保证金或现金未确认。
- 仓位上限：由仓位引擎计算，禁止融资追高。
- 失效条件：AI主线转弱、流动性反杀、账户保证金风险上升。

### SBI账户模板

- 买入触发条件：日股AI链或金融保险链回调后仍保持趋势，日元/BOJ变量未恶化，买付余力和未约定订单已确认。
- 不买入条件：9984/6857/8035连续高开、估值偏贵、日元剧烈波动、未约定订单未确认。
- 仓位上限：由仓位引擎计算，不突破日股硬上限。
- 失效条件：日股子周期转弱、BOJ超预期、个股跌破趋势。

### Bitflyer账户模板

- 买入触发条件：BTC/ETH回调但链上结构未破，ETF Flow未持续流出，日元现金和挂单已确认。
- 不买入条件：BTC脉冲上涨、杠杆/资金费率过热、交易所余额风险上升、账户余额或挂单待确认。
- 仓位上限：由仓位引擎计算，加密硬上限优先。
- 失效条件：BTC结构转弱、稳定币供应下降、ETF Flow连续流出。

## 4. 与日报 Delta 卡衔接格式

日报输出：
- 标的/方向。
- 信号等级A/B/C。
- 买入触发条件是否满足。
- 不买入条件是否触发。
- 仓位引擎是否允许计算。
- 用户人工确认事项。

## 5. 真实示例

### 美股示例：NVDA

回调到估值可观察区间，AI Narrative未破，资金流稳定 → B级或A级候选。
若高开冲高且Gamma/CTA过热 → 不买入。

### 日股示例：8035

日股AI链趋势未破但连续高开 → B级观察，不追高。
回调后日元风险缓和、买付余力确认 → 可升级。

### A股示例：半导体设备

政策催化出现且估值合理 → B级观察。
若仅题材脉冲、成交过热、账户字段待确认 → C级，不买入。

### 加密示例：BTC

BTC回调但ETF Flow未持续流出、稳定币供应未恶化 → B级观察。
若资金费率过热、价格脉冲 → 不买入。
