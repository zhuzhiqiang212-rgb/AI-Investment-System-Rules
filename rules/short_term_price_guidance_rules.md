# 短线价格指导规则

更新时间：2026-06-06 13:27:47 +09:00

## 1. 模块定位

短线价格指导只用于：

- 观察短线回调机会
- 建立小仓位参考区间
- 判断是否值得等待 dip buy
- 辅助账户动作卡

禁止用于：

- 自动下单
- 精确挂单
- 无条件加仓
- 重仓抄底
- 替代券商实时行情

## 2. 价格来源规则

短线价格必须来自：技术支撑区间、前低 / 缺口 / 均线 / VWAP、波动率区间、财报或CPI事件后的回调位置、用户输入资料中的价格观点、公开延迟行情辅助。

不能来自：单纯拍脑袋、单日涨跌情绪、只有价格没有逻辑的观点。

## 3. 必须包含逻辑论证

每一个价格必须回答：为什么是这个价、这个价对应什么支撑、什么情况下才有效、什么情况下失效、适合已有仓还是新仓、是小仓/观察仓/不能动、是否受 CPI / 非农 / 利率 / VIX / BTC / SOXX 影响。

## 4. 仓位规则

短线价格卡只允许：小仓、条件式观察、分批、等确认。

禁止：一次性重仓、无条件追高、恐慌性减仓、跌到价格就自动买。

## 5. 人工确认规则

每个价格卡必须写：实际操作前必须人工核对券商实时价格、当前持仓、成本、现金/购买力、已有挂单、是否现物/信用/保证金、数量是否超出资金或持仓。
