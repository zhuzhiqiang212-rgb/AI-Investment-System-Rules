# 财报三指标红黄绿规则

更新时间：2026-06-05 17:26:27 +09:00

## 绿色

- FCF强。
- 资产负债表健康。
- 毛利率高且稳定。
- 适合作为核心观察或长期持有候选，但不等于任何价格都可以买。

## 黄色

- 三项中一项偏弱，或行业周期性较强，或数据不够新。
- 适合持有观察，不适合无脑追高。

## 红色

- FCF弱或持续为负。
- 负债压力大。
- 毛利率明显恶化。
- 只能高风险观察，不作为核心仓。

## 特别规则

1. AI核心质量股：MSFT / NVDA / META / GOOGL / AMZN / AVGO / TSM / ASML，重点看FCF、毛利率和资本开支压力。
2. 高Beta/加密链：COIN / MSTR / IREN，额外看BTC相关性、融资压力、现金流波动和资产负债表风险。
3. 日股AI链：9984 / 6857 / 8035，区分投资控股/估值波动、半导体设备周期、订单/库存/毛利率变化。
4. 金融保险稳定器：8750 / 8766，重点看利率环境、资产负债久期、承保/投资收益和资本充足度。
5. 数据不足：明确写“数据不足，不用于确定动作”，不得用价格、题材或旧财报替代。
