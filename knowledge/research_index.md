# research_index.md
# V6 研报索引

用途：记录所有已生成研报的编号、类型、主题、时间、状态、路径与是否回写日报。
SSOT：GitHub `knowledge/research_index.md`
注意：本文件只保存索引，不保存研报正文。

---

## 状态体系（四状态）

| 状态 | 含义 | 使用条件 |
|---|---|---|
| Draft | 已生成草稿，尚未验收 | 研报初稿完成但未进入正式阅读 |
| Review | 待 ChatGPT / 用户验收 | 内容可读，但仍需确认是否发布 |
| Published (Beta-1) | 已发布试运行版 | 已可作为用户阅读与日报引用依据 |
| Archived | 历史归档 | 超过当前使用窗口，仅作为历史资料 |

## 使用规则

1. 研报正文不进入 GitHub，只进入 Drive / 桌面研报目录。
2. GitHub 只同步本索引，方便长期追踪。
3. Published (Beta-1) 表示可以阅读和引用，但仍不代表自动下单。
4. 任何研报结论进入日报时，只能作为辅助证据，不能替代系统独立判断。
