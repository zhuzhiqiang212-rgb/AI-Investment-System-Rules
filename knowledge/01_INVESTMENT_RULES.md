## 零、核心目标体系（V6.2正式版）

### 目标体系正确表述

年化40%是挑战目标，不是收益上限。

```
年度目标体系:

基础目标: 不亏损并持续增长
挑战目标: 年化40%（月度3.33%）
趋势强时: 允许60-80%，积极进攻
趋势极强时: 允许100%+，全力争取
```

### 禁止以下行为

禁止把40%当作收益上限
禁止趋势明确时以"稳健"为由放弃机会
禁止A级信号时仍只建半仓
禁止用"可能回调"否定A级信号
禁止以40%为目标自我设限，不争取超额机会

### 目标导向的系统行为

月度目标进度落后时:
  系统主动提示可提高风险敞口（在规则允许范围内）
  不沉默等待

强趋势窗口期时:
  系统主动输出进攻建议
  不只说"观察等待"

BULL A级信号时:
  系统输出具体加仓标的、账户、条件
  不输出模糊动作

## 二、进攻信号识别体系（V6.2新增）

### A级信号：主动出击

条件（全部满足）:
  总周期 BULL_MID 且置信度A级
  至少3个子周期支持（AI股/日股/加密/A股）
  无重大利率事件在未来3日内
  账户仓位未达上限

系统行为:
  日报第1层输出具体加仓建议（账户+标的+条件）
  触发进攻机会研报

### B级信号：埋伏机会

条件（全部满足）:
  总周期 BULL_MID 且置信度B级
  主线子周期未破（回调但趋势在）
  回调幅度在合理范围内

系统行为:
  日报标注"发现埋伏机会"
  触发个股深度研报或趋势轮动研报
  不立即加仓，等待B→A升级确认

### C级信号：只观察

条件: 置信度C级 或 周期为TRANSITION/UNKNOWN/BEAR

系统行为:
  日报输出"观察，不操作"
  不触发进攻类研报

## 三、强趋势窗口期识别（V6.2正式版）

### 定义（全部满足才触发）

- 总周期 BULL_MID 且置信度A级
- 4维全部共振（VIX<18 / 10Y<4.5% / SPX>+0.5% / BTC>+1%）
- SPX连续3日上涨
- VIX低于18且持续下行
- 无重大利率事件在未来5日内

### 强趋势窗口期内的系统行为

仓位: 建议接近硬上限（富途美股至45%, SBI日股至27%）
止盈: 切换为底仓持有模式（不执行+15%机械止盈）
日报: 明确标注"当前为强趋势窗口期"
研报: 自动触发进攻机会研报

### 退出强趋势窗口期

VIX突破25 或 SPX单日跌>3% → 立即退出，恢复防守

## 四、仓位管理规则

### 硬上限（任何周期下不可突破）

| 资产类别 | 硬上限 |
|---------|-------|
| 美股（IBKR+富途） | 总资产 50% |
| 日股（SBI） | 总资产 30% |
| A股（富途A股） | 总资产 30% |
| 加密（BF） | 总资产 10% |
| 现金/避险 | 最低 10% |

### 动态仓位规则

| 市场状态 | 建议仓位区间 |
|---------|------------|
| 强趋势窗口期 | 接近硬上限（-5%以内）|
| BULL_MID A级 | 70-80% |
| BULL_MID B级 | 50-70% |
| TRANSITION B级 | 20-40% |
| BEAR / C级 | 0-20% |

## 五、风险管理规则

止损: 单笔最大亏损账户净值3%，进场时确定，不事后移动
EMERGENCY MODE触发: 股指单日跌>5% / VIX单日跳升>30% / 用户手动宣布
EMERGENCY MODE规则: 周期降BEAR / 买入全停 / 现金强制30%

## 六、买卖纪律

### 买入条件（全部满足）
1. 周期BULL_MID或BULL_EARLY
2. 置信度A级（B级可埋伏小仓试探）
3. 信号A或B
4. 仓位未达上限
5. 用户人工确认

### 禁止买入（任意一条）
置信度C级 / TRANSITION/UNKNOWN/BEAR /
标的5日涨>12% / 回撤>10% / EMERGENCY中 / 数据失败≥3项

### 止盈规则

| 场景 | 止盈策略 |
|------|---------|
| 标准场景 | +15%止盈50% / +30%止盈剩余50% |
| 强趋势窗口期 | 底仓持有，不执行机械止盈 |
| B级埋伏仓 | +10%止盈，不持有底仓 |
| 周期失效时 | 不论盈亏强制全部止盈 |

## 七、周期分析（4维判断）

| 维度 | BULL | TRANSITION | BEAR |
|------|------|-----------|------|
| VIX | <18 | 18-23 | >23 |
| 10Y美债 | <4.0% | 4.0-4.7% | >4.7% |
| SPX日涨跌 | >+0.5% | -1%~+0.5% | <-1% |
| BTC日涨跌 | >+1% | -2%~+1% | <-2% |

置信度: 3/4或4/4=A | 2/4=B | 1/4=C | 数据不足<3维降一级

## 八、子周期框架

总周期: 宏观流动性（VIX/美债/SPX/BTC）

子周期（各自独立判断，禁止用总周期覆盖）:
  AI股子周期: SPX走势 + NVDA/TSM浮盈状态 + AI叙事强度
  加密子周期: BTC走势 + MSTR/COIN浮盈对比（是否分化）
  日股子周期: BOJ事件 + 日元走势 + SBI账户整体浮盈
  A股子周期: 北向资金（待接入）

子周期"独立强势"规则:
  某子周期信号强于总周期时 → 标注"独立强势"
  允许在该子周期板块小仓埋伏

## 九、归因原则（M1-M6）

M1: 周期定位错误 | M2: 资产配置错误 | M3: 买入触发错误
M4: 仓位计算错误 | M5: 止盈规则错误 | M6: 用户执行偏离

未交易归因（V6.2更新，六类）:
  A=触发未买 / B=应减未减 / C=违规买 / D=应止未止
  E=强趋势窗口期仓位过轻，错失超额机会
  F=发现B级埋伏机会但未建立埋伏仓（V6.2新增）

最低样本: 触发器≥10次 / 周期≥8次才可统计
不足时输出: "样本不足（N次/最低M次），继续观察"
