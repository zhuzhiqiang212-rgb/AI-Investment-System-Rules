## 一、日报标准（auto_briefing.py 自动生成）

### 输出文件
latest_daily_report.md（今天发生了什么）
latest_direction_card.md（主线方向是否变化）
latest_risk_card.md（最大风险是什么）
latest_execution_card.md（今天该做什么）

### 日报内容规范

#### latest_daily_report.md
```
# 今日日报 — YYYY-MM-DD
生成时间：YYYY-MM-DD JST

[昨日验证行]
昨日判断（YYYY-MM-DD）：[周期] [置信度]级 — [layer0]
今日实际：SPX [X]% / VIX [X]
验证结论：[防守方向成立/进攻方向成立/市场震荡/数据缺失]

## 今日市场变量
VIX：[值] 10Y美债：[值]% SPX：[值]% BTC：[值]%

## 今日结论
[layer0]
周期：[标签] 置信度：[A/B/C]
```

#### latest_direction_card.md
```
# 方向卡 — YYYY-MM-DD
## 当前方向
周期：[标签] 置信度：[A/B/C]
方向：[layer0]

## 失效条件
[基于周期的失效条件]

## 方向小结
置信度[X]级——[是否可按此执行]
```

#### latest_risk_card.md
```
# 风险卡 — YYYY-MM-DD
## 最大风险
[layer2全部条目]

## 风险触发阈值
VIX当前：[值]（警戒：>25）
SPX当前：[值]（警戒：单日<-2%）

## 数据源状态
VIX：[OK/DATA_GAP] 10Y：[OK/DATA_GAP]
SPX：[OK/DATA_GAP] BTC：[OK/DATA_GAP]
```

#### latest_execution_card.md
```
# 执行卡 — YYYY-MM-DD
## 唯一动作
[layer1]

## 四账户边界
富途：[动作]
IBKR：[动作]
SBI ：[动作]
BF  ：[动作]

## 执行前必须确认
□ 券商实时行情
□ 当前持仓和成本
□ 可用现金
□ 未成交挂单

NEXT_OWNER  : 用户
NEXT_ACTION : [基于结论的具体建议]
NEXT_THREAD : AI投研总控台 + 正式日报生产
```

### 日报质量标准
- 第0层和第1层不得自相矛盾
- TRANSITION/UNKNOWN/BEAR时第1层必须是"观察"
- 第2层至少两条，第二条含当日具体数据
- 报表日期必须为当日（旧日期=FAIL）

## 三、月报标准（手动生成，每月末）

使用 monthly_return_tracking_v1.md 模板填写：

必填项：
- 月初/月末总资产
- 本月实际收益率
- 与3.33%月度目标的差距
- 四账户分解贡献度
- 六模块差距归因（必须归因到具体模块）

禁止项：
- 样本不足时输出"策略有效/失效"结论
- 归因结论只写"判断失误"而不指向具体模块

触发G-10策略迭代的条件：
- 连续3个月收益率低于2%
- 任一月份亏损超过-5%
- 同一模块连续3个月为主要拖累
