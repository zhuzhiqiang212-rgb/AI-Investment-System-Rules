# AI_Investment_System

## 1. 系统是什么

这是用户的 AI 投研闭环系统，用于：

- 生成用户版投研日报
- 管理四账户动作
- 跟踪 AI 超级周期
- 跟踪宏观、资金流、ETF Flow、Positioning/Gamma/CTA
- 跟踪财报三指标、护城河、短线价格
- 建立证据链和明日 PDCA
- 做质量验收和最终收口

本系统不是自动交易系统，不直接下单，也不替代用户对账户和交易的最终确认。

## 3. Google Drive 存什么

Google Drive 保存：

- 日报 HTML
- 账户截图
- PDF / 视频转文字 / 湖水 / 老雷资料
- 验收包
- 历史报告
- 账户快照
- 用户上传资料

## 4. GitHub 存什么

GitHub 只保存：

- 系统脚本
- 规则文件
- Skill 文件
- 模板
- 说明文档
- 目录结构
- 质量标准

GitHub 不保存：

- 账户截图
- 大型 PDF
- 视频音频
- 个人隐私资料
- 每日报告大文件
- 历史归档大文件

## 5. 角色分工

- 用户：最终确认者
- ChatGPT：总设计 / 质检 / 吹哨 / 指令生成
- Codex：工程执行者
- Cursor：代码维护工作台
- Claude：备用军师 / 第二质检
- GitHub：系统代码保险柜
- Google Drive：资料和报告仓库
- Skill：AI 岗位说明书
