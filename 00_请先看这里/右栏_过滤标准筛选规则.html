过滤标准 / 筛选规则 · 右栏底子

右栏最后一块 ｜ 从一堆股票筛到"值得看"的完整漏斗规则 ｜ Codex拿这套规则去扫描
依据：蓝图清单⑤筛选层+已建三关(硬性/软性/估值)+护城河框架+个股研究。这是"尺"——Codex拿这套规则去板块里扫，筛出机会标的。规则固定，扫出的名单每天动态变。
全市场 → 第1关 硬性 → 第2关 软性 → 第3关 估值 → 第4关 护城河 → 第5关 个股研究 → 值得看
从粗到细，一关关刷。先刷掉方向不对的，再从对的里挑好的、看价格、看质地、看深度。

第1关 · 硬性过滤（一票否决）

刷掉：方向不对的。不管多好，方向不对直接出局。

规则：这个标的，在不在"今天激活的板块"上？（激活＝证据链今天推出的方向所指的板块）

在 → 过关；不在 → 一票否决、受检（哪怕它再便宜再优质）。这一关保证"只在对的方向里挑"，是自上而下的闸。

第2关 · 软性过滤（位置/龙头/量能）

从方向对的里，挑位置好的。

均线位置（看技术位置）：

看什么怎么判
200日均线（年线）在年线上方＝长期趋势健康；跌破＝趋势坏，警惕
50日均线中期趋势；股价回踩50日不破＝低吸位
20日均线短期趋势；短线强弱

好的位置：年线上方 + 回调到50日附近低吸（不追高、不接飞刀）。龙头：优先板块龙头。量能：突破要放量确认，缩量突破不算数。

第3关 · 估值（贵不贵）

位置好的里，看价格值不值。

规则：按标的类型用对应估值模型，算"该值多少"，和现价比：

类型估值模型
成长股成长PE / PEG
控股/投资公司NAV净资产（如软银）
周期股周期PE（跨周期看）
医药DCF/管线折现（如第一三共）
保险EV内含价值（如东京海上）
加密相关mNAV（如MSTR）

便宜/合理 → 过；明显贵 → 记下但不追，等回调。

第4关 · 护城河（质地·能不能长拿）

便宜的里，看是不是真好公司（不是便宜有便宜的道理）。

规则：五维打分（品牌/网络效应/成本/转换成本/专利），每维宽2/窄1/无0，总分7-10宽、4-6窄、0-3无。宽护城河+便宜＝最优质机会。（详见护城河分析框架）

第5关 · 个股深度研究（最后的细看）

过了前4关的少数，值得深挖了。

规则：财报（营收/利润/现金流趋势）、业务/管线（增长点、看得见的未来）、管理层（靠不靠谱）、竞争格局（对手强不强）。此关＝理解岗深度研究，建个股档案。目前待建。

和板块地图左右对应（一条流水线）：
右栏板块地图（哪些板块·状态）→ 右栏本层过滤规则（这把尺）→ Codex拿尺去"今天激活的板块"里扫 → 左栏机会池（扫出的具体标的·每天变）。
板块地图告诉"去哪个板块扫"，本层告诉"用什么标准筛"，两者合起来才扫得出标的。规则固定、标的动态。
