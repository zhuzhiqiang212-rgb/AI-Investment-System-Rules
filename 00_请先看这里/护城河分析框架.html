护城河是什么、为什么放在估值之后

护城河＝一家公司"能不能长期挡住对手、持续赚钱"的本事。城堡（生意）再好，没有护城河（壁垒），迟早被对手攻进来、利润被抢光。

为什么排在估值之后：前面估值算了"现在贵不贵"，但便宜的公司可能是"便宜有便宜的道理"（要被淘汰）。护城河这一关就是问：这家公司值得长期拿吗？它的赚钱能力是不是稳的、护得住的？便宜＋有宽护城河，才是真正的好机会。

五维护城河（巴菲特式，每维打分）

1. 品牌

客户是不是"认这个牌子、愿意多付钱"？
例：可口可乐、苹果——牌子本身就是壁垒。你持仓里的任天堂（IP品牌）也算。

2. 网络效应

用的人越多、它越值钱、越难被替代？
例：微信、信用卡网络、交易所（Coinbase）——用户互相锁住。

3. 成本优势

它能比对手更便宜地造出同样东西？
例：台积电的制程规模、规模化生产——对手做不到这么便宜。

4. 转换成本

客户想换到对手那儿，麻烦/代价大不大？
例：微软的Office/系统——企业换掉它成本极高，只能一直用。

5. 无形资产/专利技术

有没有别人拿不到的专利、牌照、技术垄断？
例：英伟达的AI芯片技术护城河、第一三共的抗癌药专利、爱德万的设备技术。

怎么打分（每维+总评）

每一维打分意思
宽（2分）这一维壁垒很强、对手难攻
窄（1分）有一点壁垒，但不牢
无（0分）这一维没壁垒

总评（五维加总，满分10）：

总分护城河含义
7-10宽护城河能长期赚钱、值得长拿
4-6窄护城河有壁垒但要盯着，别人可能追上
0-3无护城河纯拼价格/风口，不值得长拿

★ 补充 · 公用事业专用护城河评法（受监管电/水/气）· 2026-07-22 增补

为什么要单开一套：上面那套"巴菲特五维"（品牌/网络/成本/转换/专利）是给制造·科技·消费公司用的。用它去量受监管公用事业（电力/自来水/燃气公司），会系统性打低分——因为公用事业的真护城河根本不在这五维里。

实证：机器用通用五维给你候选里的 D（Dominion 弗吉尼亚电力）和 PEG（PSEG 新泽西电力）打分，两只都只有 3 分＝"无护城河"。这是错判。受监管电网是巴菲特最偏爱的宽护城河生意之一（他旗下 BHE 就是干这个的）。问题不在这两只公司，在"尺"用错了。

公用事业的护城河来自哪——政府发的垄断牌照：某片地区的供电权只给你一家，别人不许进来抢；换来的是你得普遍供电、且赚多少受监管机构管着。所以它护城河极宽，但回报有天花板。据此定专用五维：

公1. 监管特许经营权（垄断牌照）

是不是政府授予的独家服务区？别人能不能进来抢这片地的生意？
宽＝法定独家特许、排他性强；窄＝部分放开/有竞争性零售；无＝纯批发/无特许。这是公用护城河的第一根。

公2. 高效规模 / 自然垄断（电网/管网）

这片地的输配电网是不是"只容得下一家"？再建第二张网划不划算？
宽＝区域唯一电网、重复建设不经济（自然垄断）；窄＝部分区域重叠；无＝可竞争。晨星称这为 efficient scale——通用五维里缺的正是它。

公3. 受监管回报的确定性（rate base 质量 + 监管环境）

监管机构给不给稳定回报？环境是"配合建设型"还是"对抗压价型"？投资基数(rate base)在不在增长？
宽＝监管建设性、允许ROE稳、rate base 随投资增长（如数据中心接入）；窄＝监管偶有摩擦/滞后；无＝频繁被压价。回报天花板≈行业允许ROE普遍 9.5–10.5%（具体以各州 rate case 为准，不得编数）。

公4. 资产壁垒（电网不可复制）

要复制它这张网，得花多少钱、多少年？新对手够得着吗？
宽＝输配网+发电资产重置成本极高、建设周期几十年；窄＝资产较轻；无＝易复制。

公5. 需求刚性 / 必需品（扣分看侵蚀）

用电是刚需吗？有没有东西在侵蚀它——自发电、分布式光伏、去监管？
宽＝需求刚性且增长（AI数据中心用电爆发）；窄＝需求平/有侵蚀；无＝被替代明显。

打分与总评：同通用尺——每维 宽2/窄1/无0，总分 7-10 宽 / 4-6 窄 / 0-3 无，便于接同一漏斗第4关。

★★ 天花板诚实注（必读·防误导）：公用事业"宽护城河"＝利润稳、别人抢不走、下行保护强、分红可预测——但不等于高成长。它的回报被监管封顶（允许ROE就那么高），是"债券替代型"的防御/收息仓，不是英伟达那种进攻仓。给它宽护城河，意思是"稳得住值得配一点收息"，不是"会大涨"。别拿护城河分高当成"该重仓博上涨"。
接口规则（什么股走这套）：主营受州/联邦公用事业委员会监管的输配电/气/水、收入主要来自 rate-regulated 业务的公司 → 走"公用专用五维"；其余一律走通用五维。混合型（如含大量非监管电力零售/发电的，参考 CLAUDE.md 5.5 坑：NRG 毛利率 13.6% 是零售商属性、非电厂）→ 拆开看、就低不就高，标"混合·待理解岗定主业"。

D / PEG 示范打分（架构师初判 · 档位纠偏 · 具体数字待理解岗核实各州最新 rate case，不得编）：

标的公1特许公2规模公3回报公4资产公5需求总分/档
D Dominion·弗吉尼亚宽2宽21~2 待核宽2宽2（数据中心用电爆发）9~10 宽护城河
PEG PSEG·新泽西宽2宽21~2 待核宽21~2（负荷增长弱于弗州）8~9 宽护城河

对比：通用五维把这两只判成 3 分"无护城河"（错）；公用专用评法纠正为"宽护城河（但收息型·非成长）"。公3 回报维给区间 1~2、不写死，是因为具体允许ROE/rate base/最新 rate case 结果须理解岗查证后落定——按生产铁律，缺数标"待核"，不估算放行。

怎么接进筛选漏斗（第4关的作用）

一个标的过了硬性（在方向上）+软性（位置好）+估值（不贵）后，到护城河这关：

· 宽护城河 + 便宜 → 最优质机会，重点关注
· 窄护城河 + 便宜 → 可以，但盯紧壁垒会不会被攻破
· 无护城河 + 便宜 → 便宜有便宜的道理，谨慎（可能在被淘汰）

诚实边界：护城河是"定性判断"，不是算出来的数。前期由理解岗(Claude)结合公开资料+个股研究打分，标注置信度；不假装是精确公式。
